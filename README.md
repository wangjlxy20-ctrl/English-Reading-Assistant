# English Reading Assistant

English Original Reading Assistant Based on  Spring Boot + MyBatis + MySQL


## Tech Stack

Backend:
- Spring Boot
- MyBatis
- MySQL
- Maven


## Features

- Book management
- Chapter reading
- Reading progress tracking


## Current API

GET /books

GET /chapters/book/{bookId}

GET /records/{userId}

POST /records


## Future

- Vue3 frontend
